Bem-vindo ao repositório do meu Portfólio Pessoal! Este projeto foi desenvolvido para apresentar meus projetos, habilidades e experiências na área de desenvolvimento de software, servindo como um hub central para meus trabalhos acadêmicos e pessoais.

---

## 🚀 Sobre o Projeto

Este é um site estático, responsivo e interativo, construído com tecnologias fundamentais da Web sem o uso de frameworks pesados, demonstrando domínio sobre a base do Front-end.

### ✨ Funcionalidades Principais

*   **Internacionalização (i18n):** Sistema robusto de tradução para **Português (BR)**, **Inglês** e **Espanhol**. O idioma escolhido é salvo no navegador (`localStorage`) e persiste durante a navegação entre páginas.
*   **Temas Claro e Escuro:** Alternância de temas (Dark/Light Mode) com persistência de preferência do usuário.
*   **Design Responsivo:** Layout adaptável para dispositivos móveis e desktops (Mobile-First approach).
*   **Navegação Dinâmica:** Menu mobile interativo e rolagem suave (Smooth Scroll) para seções.
*   **Showcase de Projetos:** Páginas detalhadas para projetos específicos:
    *   *Metaverse*: Projeto focado em Realidade Virtual.
    *   *SCGA*: Sistema de Controle e Gerenciamento de Ambientes (TCC).
    *   *Outros Projetos*: Listagem de exercícios e projetos menores.
    *   *Fatec*: Repositório com todas as atividades e projetos desenvolvidos na Fatec.

| Tecnologia | Detalhes |

## 🚦 Como Executar Localmente

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/GustavoPereira-Dev/Portfolio.git
    ```
2.  **Abra o projeto:**
    *   Como o projeto utiliza **Módulos JavaScript** (`type="module"`), é recomendável usar um servidor local (como a extensão *Live Server* do VS Code) para evitar bloqueios de política de segurança (CORS) do navegador ao abrir o arquivo `index.html` diretamente.
